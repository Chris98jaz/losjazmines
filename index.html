Ejemplo de Formateo en HTML

Este es un ejemplo de cómo puedes aplicar diferentes estilos de texto en HTML. Aquí hemos usado color, tamaño de fuente, y alineamiento de texto.

Puedes aplicar negrita para resaltar texto importante, cursiva para enfatizar palabras, y subrayado para destacar información crucial.

Además, este párrafo está alineado de forma justificada, lo que significa que el texto se extiende uniformemente hasta ambos márgenes, creando un borde recto a la izquierda y a la derecha. Esta técnica es común en documentos más formales como artículos y ensayos.

A continuación, mostramos una lista de las tecnologías web más populares:
